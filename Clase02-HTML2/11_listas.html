Listas desordenadas

Las listas desordenadas nos muestran los elementos de la lista precedidas por una pequeña viñeta.
Las listas desordenadas se representan mediante el elemento UL.

  • Elemento 1
  • Elemento 2
    • ...
  • Elemento N

Cambiando la viñeta

Las viñetas que se muestran en las listas desordenadas se pueden cambiar. Por defecto son de tipo disc,pero hay algunas alternativas más:

  • Esta es una viñeta del tipo circle.
  • Esta es una viñeta del tipo square.
  • Esta es una viñeta del tipo disc.

Listas ordenadas

Las listas ordenadas son aquellas que nos muestran los elementos de la lista en orden. Para representar el orden tendremos los elementos numerados. Es decir, cada uno de los elementos irá precedido de un número o letra que establezca su orden.
Las listas en HTML ordenadas se representan mediante el elemento OL.

  1. Este es el primer ítem de la lista.
  2. Este es el segundo ítem de la lista.
  3. Otro elemento...
  4. Y otro más.

Las listas ordenadas pueden comenzar de números diferentes a 1.
Para ello usamos el atributo value.

  1. Este será el número 20.
  2. Este será el 21.
  3. Este será el 22. Y así sucesivamente.

Cambiando los números

Las viñetas que se muestran en las listas desordenadas se pueden cambiar. Por defecto son de tipo disc,pero hay algunas alternativas más:

  1. Esta lista enumera sus elementos..
  2. usando letras minúsculas.
  3. Hay que utilizar el atributo type="a"
  1. Esta lista enumera sus elementos..
  2. usando letras mayúsculas.
  3. Hay que utilizar el atributo type="A"
  1. Esta lista enumera sus elementos..
  2. usando números romanos (minúsculas).
  3. Hay que utilizar el atributo type="i"
  1. Esta lista enumera sus elementos..
  2. usando números romanos (mayúsculas).
  3. Hay que utilizar el atributo type="I"
