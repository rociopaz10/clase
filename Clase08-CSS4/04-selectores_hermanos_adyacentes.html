Titulo: Selectores avanzados Etiqueta: style Texto: h3+p { background-color: yellowgreen; } . Etiqueta: h3 Texto: Selector de hermano adyacente Etiqueta: p Texto: El signo + (mas) permite aplicar estilos a elementos que siguen a otros, es decir que está directamente después de otro elemento específico. Los elementos hermanos deben tener el mismo elemento padre y "adyacente" significa "inmediatamente siguiente". No puede haber ningún otro hermano que los separe o se interponga entre ellos. Etiqueta: p Texto: El color de fondo verde se aplica sólo al párrafo que está inmediatamente despues del div, y a ninguno de los otros. Etiqueta: div Etiqueta: p Texto: Parrafo 1, dentro del div. Etiqueta: p Texto: Parrafo 2, dentro del div. Etiqueta: p Texto: Parrafo 3, inmediatamente despues del div. Etiqueta: p Texto: Parrafo 4, después del div, pero con el parrafo 3 en medio. Etiqueta: div Etiqueta: p Texto: Parrafo 5, dentro de otro div. Etiqueta: a href="#" Texto: Un link a ninguna parte. Etiqueta: p Texto: Parrafo 6, después del div, pero con el link en medio.
