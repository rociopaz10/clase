Titulo: Transiciones Etiqueta: style Texto: /*------------- TRANSICIONES -------------*/ /*Truco: Si nos fijamos bien, este estilo se aplica sólo al mover el ratón sobre el enlace (transición de entrada). Sin embargo, si movemos el ratón fuera del enlace, no se produce transición sino que realiza el cambio de forma brusca. Si movemos las propiedades de transición al primer bloque (a {}), se aplican tanto en las transiciones de entrada como en las de salida.*/ a { background: #DDD; color: #222; padding: 2px; border: 1px solid #AAA; } a:hover { background: #FFF; color: #666; padding: 8px 14px; border: 1px solid blue; transition-property: all; /*a todas las propiedades nombradas*/ transition-duration: 0.5s; /*con una duración de 0.5s*/ transition-timing-function: ease-in; /*inicio lento, luego normal*/ } Etiqueta: h3 Texto: Transiciones Etiqueta: p Texto: Las transiciones CSS le permiten cambiar los valores de una propiedad, durante un período determinado. Se basan en un principio muy básico: conseguir un cambio de estilo con un efecto suavizado entre un estado inicial y un estado final. Etiqueta: a href="" Texto: Enlace
