Atributos de página - Enlaces

Podemos modificar la apariencia de todos los enlaces que aparecen en nuestra página desde la etiqueta body.

Se aplican a todos los enlaces del documento.

link: color del link sin visitar.
alink: color del link seleccionado (cuando estamos haciendo el click)
vlink: color del link visitado.
